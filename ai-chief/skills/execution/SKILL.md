# Execution Skill

## 何时使用
- 需要直接产出任务结果时
- 需要把任务转成可执行结论与行动项时

## 输出模板
# Result
## Conclusion
## Evidence
## Risks
## Next Actions
## Confidence (0-1)
## Need Human Decision (Yes/No + reason)

## 规则
- 信息不足必须写清假设
- 置信度低于阈值必须升级人工
- 证据必须可追溯
